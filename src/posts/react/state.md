# 组件状态



1. state 是 继承来的一个属性  state 是一个对象
2. state 初始化，必须在 constructor 中，this.state = {}
3. state 改变， 必须使用 this.setState({}), 参数和原来的 state 混合， 得到新的state
4. 一旦调用了this.setState，会导致当前组件重新渲染
