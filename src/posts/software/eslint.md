# eslint

1. vscode extensions 
点击左层sidebar的extensions
2. 升级eslint
升级到预发布版本
3. eslint setting
找到format enable，勾选复选框
4. 重启eslint
command + shift + p 
restart eslint server
5. 设置默认格式化工具
右键，找到format with ...
选择最后一行配置
选择eslint
